
# Setup & Running

## Local
1. python -m venv venv
2. source venv/bin/activate
3. pip install -r agent_server/requirements.txt
4. export GOOGLE_API_KEY=your_key
5. python agent_server/agent_server.py

## Kaggle
1. Upload the repository as a ZIP to Kaggle or add files manually
2. Set Kaggle Secret 'GOOGLE_API_KEY' under Add-ons -> Secrets
3. Run the notebook in /notebooks/AIMMR_Final_Notebook.ipynb
