This repository contains the AIMMR (AI Mental Model Reconstructor) project skeleton.
It includes core Python modules, a FastAPI agent server, and a test notebook.

## Quick start
1. Clone repository
2. Set environment variable: `export GOOGLE_API_KEY=your_key`
3. Run server: `python agent_server/agent_server.py`
4. Open `agent_server/index.html` in browser and test

## Structure
- aimmr_core/: core modules (PatternAnalyzer, CognitiveModelBuilder, MisconceptionDetector, PersonalizedTeacher)
- agent_server/: FastAPI server and simple UI
- notebooks/: test notebook
